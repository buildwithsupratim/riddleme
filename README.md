# 🧩 RiddleMe — AI Learning Through Riddles

RiddleMe is an AI-powered learning notebook for kids that teaches through fun riddles, playful questions, and simple explanations.

Instead of only giving direct answers, RiddleMe helps children think, guess, imagine, and remember.

## ✨ Features

- Teaches through riddles after every lesson
- Uses simple language for children aged 6–12
- Makes learning more interactive and fun
- Encourages curiosity and problem-solving
- Gives short explanations before each riddle
- Supports multiple boards, classes, and subjects
- Includes teacher personas for different teaching styles
- Uses RAG to provide curriculum-aware answers
- Lets children choose only the topics they want to learn
- Built with a colorful, kid-friendly interface

## 🧠 Why Riddles Help Kids Learn

Riddles are not just fun — they also improve learning.

They help children:

- Think more deeply
- Improve memory recall
- Build logic and reasoning skills
- Stay curious and engaged
- Learn faster through play
- Remember concepts for longer
- Develop confidence when solving problems
- Enjoy lessons instead of feeling bored

## 🎭 Example Learning Style

Lesson:

> Plants need sunlight, water, air, and soil to grow.

Riddle:

> I help plants grow but I am not food.  
> I shine in the sky every day.  
> What am I?

Answer:

> The Sun ☀️

This makes learning feel more like a game than a classroom lesson.

## 📚 Curriculum Support

RiddleMe supports:

- Multiple boards
- Multiple classes
- Multiple subjects
- Topic-by-topic learning
- Curriculum-aligned lessons
- FAISS-based semantic search
- Knowledge retrieval using RAG
